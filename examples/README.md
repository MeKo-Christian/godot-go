# Examples

This directory contains standalone Godot demo projects backed by godot-go.

- `hello_world/`: Minimal GDExtension that prints a message from Go.
- `physics_2d_bouncing_ball/`: RigidBody2D demo with a bouncing ball.
- `input_handling/`: Keyboard and mouse input logging demo.
- `signal_demo/`: Custom signal emission and connection demo.
- `ui_overlay_score/`: UI overlay demo with a score label.
- `actions_input_demo/`: InputMap actions + polling demo.
