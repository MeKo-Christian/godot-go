# Camera + Viewport Demo

This demo validates Camera2D following behavior and viewport size handling.

## What it shows

- Camera2D follows a moving target node with position smoothing enabled.
- A label updates when the viewport size changes.

## Run

1. Build the extension for this example.
2. Open `examples/camera_viewport/project.godot` in Godot 4.5+.
3. Run the scene and resize the window to see the label update.
